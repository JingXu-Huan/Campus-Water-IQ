# 网关微服务开发手册
## 本模块职责
* 统一入口：作为系统唯一对外入口，承载所有客户端请求

* 请求路由：根据路由规则将请求转发至对应业务微服务

* 负载均衡：在多实例服务之间进行流量分发，提升系统可用性

* 统一鉴权：校验访问凭证（Token），拦截非法请求

* 基础安全控制：实现限流、防刷、跨域等通用安全策略
## 注意：
* 关于鉴权：在网关层面，我们只检查：
    * Token 在不在
    * Token 合不合法
    * Token 过没过期
    * 这个接口要不要放行
* 不检查（放到Auth-Service）：
    * 登录
    * 查用户表
    * 判断密码对不对
    * 动态算权限
